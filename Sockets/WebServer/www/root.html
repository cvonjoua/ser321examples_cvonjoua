You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- enter 2 integer variables num1 & num2, which will get multiplied and returned
  •         Example: root/multiply?num1=10&num2=10
  • /github -- when username specified, will supply list of all repositories and information for each repository
    •         Information provided: repo full name; repo ID; and repo owner.         Example: root/github?query=users/[USERNAME]/repos
  • /charcount -- enter 2 strings str1 & str2, and total number of characters will be returned to you minus the white space
    •         Example: root/charcount?str1=Hello&str2=World - will return 10 characters.
  • /gradecheck -- enter the grades for 2 assignments num1 & num2 and it will give you your average grade and the letter grade associated
    •         Example: root/gradecheck?grade1=100&grade2=90 - Will return 95.0% which is an A.
