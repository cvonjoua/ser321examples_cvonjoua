You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /charcount -- enter 2 strings str1 & str2, and total number of characters will be returned to you minus the white space
  • /gradecheck -- enter the grades for 2 assignments num1 & num2 and it will give you your average grade and the letter grade associated
